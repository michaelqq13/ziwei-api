📋使用方法

  1. 準備圖片 - 將您的圖片放入 user_images 資料夾
  2. 執行生成器 - 運行 python demo_custom_buttons.py
  3. 配置按鈕 - 自定義按鈕文字、顏色和特效
  4. 生成選單 - 自動創建完整的 Rich Menu 圖片
  5. 上傳使用 - 將生成的圖片上傳到 LINE Bot
# 基本使用範例 from app.utils.custom_button_generator import CustomButtonGenerator generator = CustomButtonGenerator() # 配置按鈕 configs = [ { 'image_path': 'user_images/my_icon.png', 'button_text': '我的功能', 'text_color': (255, 255, 255), 'add_background': True, 'add_shadow': True } ] # 生成按鈕 button_paths = generator.create_button_set_from_images(configs) # 整合到 Rich Menu menu_path, button_areas = generator.integrate_with_rich_menu( button_paths, button_configs, "custom_menu" )

🚀 批量處理

生成的圖片完全符合 LINE Rich Menu 規格，可直接上傳使用。

🎛️進階配置

支援更多自定義選項：

# 進階配置範例 advanced_config = { 'image_path': 'user_images/special_icon.png', 'button_text': '特殊功能', 'button_size': (250, 250), # 自定義尺寸 'text_color': (255, 215, 0), # 金色文字 'add_background': True, # 星空背景 'add_border': True, # 邊框效果 'add_shadow': True # 發光陰影 } # 不同風格的按鈕 minimal_config = { 'image_path': 'user_images/simple_icon.png', 'button_text': '簡約風格', 'add_background': False, # 無背景 'add_border': False, # 無邊框 'add_shadow': False # 無特效 }
